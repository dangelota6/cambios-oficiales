Accesibilidad

En esta página podrá encontrar tanto un mapa del sitio disponible para facilitar el acceso a las principales secciones. Dentro de las acciones realizadas por OSIPTEL para el cumplimiento de las Pautas de Accesibilidad especificadas por W3C. (W3C-WAI), podemos señalar:

  • Código: se escribió en base a las Pautas de Accesibilidad especificadas por W3C. (W3C-WAI).

  • Imágenes: todas las imágenes tienen texto alternativo de acuerdo al contenido que representan.

  • Mapa del sitio: se incorporó un Mapa del Sitio para ayudar la navegación y la visualización rápida de todas las secciones de nuestro sitio web.

  • Estilos: Todas las propiedades de diseño se aplican mediante hojas de estilo CSS. Logrando que el sitio sea mucho más rápido para navegar

  • Diseño: se maquetó el sitio utilizando etiquetas <DIV>.

  • Nivel de Accesibilidad: se ha logrado obtener la validación para el nivel A de las Directrices de Accesibilidad para el Contenido Web 1.0 2.

  • Validación: se obtuvo la validación en XHTML 1.0 y CSS Versión 2. 2.

Características de visualización

El Portal de OSIPTEL está optimizado para una resolución de 1024x768. Si se tiene un monitor con una resolución mayor, puede que el tamaño de letra resulte en algunos casos pequeño. No obstante, los tamaños de las fuentes se han definido con unidades relativas para que pueda ampliar o disminuir el tamaño de la fuente desde las opciones del navegador.

Se ha comprobado la correcta visualización de este sitio Web en diferentes versiones de los principales navegadores y en diferentes plataformas. Este portal está optimizado para los navegadores tales como Microsoft Windows Internet Explorer, Mozilla Firefox, Google Chrome.

Aumentar y reducir el tamaño del texto

Las fuentes utilizadas tienen dimensiones relativas, de forma que puedes controlar su tamaño desde el navegador que estés utilizando.

Para aumentar el tamaño de la fuente, en casi todos los navegadores basta con utilizar la combinación de teclas CTRL+. Usaremos, asimismo CTRL– para disminuirla y CTRL 0 para volver al tamaño original. Con Google Chrome, además, si pinchamos en el botón “herramientas” a la derecha de la barra de direcciones, podremos utilizar su barra de zoom y su botón de pantalla completa.

Incidencias y sugerencias

Nuestra preocupación es que la información pueda llegar al mayor número posible de personas. Si mientras navegas por las páginas web tienes algún problema para acceder a los contenidos, o si quieres hacer alguna sugerencia que nos pueda ayudar a mejorar la accesibilidad web, te invitamos a utilizar el buzón de consultas y sugerencias de www.osiptel.gob.pe.

En algunos casos puede ser que las páginas se complementen con contenidos presentados en otros formatos (documentos, vídeos, animaciones, etc.). Si no dispones de los programas necesarios para acceder a estos contenidos te ofrecemos los conectores (plugins) y programas gratuitos.

Formatos utilizados

Formato PDF: La mayor parte de los documentos que se descargan de esta Web se encuentran en formato Adobe Acrobat®, el popular PDF (Portable Document Format), que se utiliza generalmente para preservar el contenido y composición de documentación y publicaciones impresas. Los documentos PDF (extensión .pdf) sólo se pueden visualizar con el lector Adobe Acrobat Reader® , que se descarga de manera gratuita del sitio Web de Adobe. Se recomienda utilizar Adobe Acrobat Reader® en su versión 6.0 o superior.

Formato ZIP: Los documentos ZIP (extensión .zip) se abren con el programa de compresión y descompresión de archivos WinZip®, aunque también existen otros programas similares que soportan este mismo formato. En el caso de que no disponga de este software, puede descargarse una versión de evaluación de WinZip.

Formato SWF: Algunos documentos SWF (extensión .swf) pueden ser visualizados directamente si tiene la versión adecuada de Flash instalada en su ordenador, Adobe ofrece un visor gratuito de archivos Flash, que se descarga desde del sitio Web de Adobe para solventar problemas de incompatibilidad.
